## Run

```bash
bun install
bun start          # or: bun run index.ts
```

Make it a global command:

```bash
chmod +x index.ts
bun link           # then run `pranav` from anywhere
```

## Keys

| Key               | Action                                  |
| ----------------- | --------------------------------------- |
| `‚ÜĎ` `‚Üď` / `j` `k` | move selection                          |
| `enter`           | open selected ship (live URL or GitHub) |
| `g`               | open selected ship's GitHub             |
| `s`               | open pranavkarthik.com                   |
| `x`               | open x.com profile                      |
| `q` / `esc`       | quit                                    |

## Layout

- **Left** ‚ÄĒ selectable list of recent ships (most recent first).
- **Right** ‚ÄĒ live detail for the selected ship: year, type, description, stack,
  awards, plus a `NOW` / `EDU` block.
- **Bottom** ‚ÄĒ keybinding hints.

## Files

- `index.ts` ‚ÄĒ the app. `mount(renderer)` builds the UI; bootstraps a real
  `CliRenderer` when run directly.
- `data.ts` ‚ÄĒ all personal content (profile, ships, experience, education).
  Edit this to update the page.
- `snapshot.ts` ‚ÄĒ renders a plain-text frame for quick visual checks:
  `bun run snapshot.ts [cols] [rows]`.
